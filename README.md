# Decodificador - Matrix

Decodificador de texto feito durante a realização do Desafio Oracle ONE.

Funções:

- Um campo de texto que aceita somente letras minúsculas e sem acento
- Botão de criptografar, que griptografa o texto de acordo com as especificações do desafio.
- Botão de descriptografar, que descriptografa o texto e reverte-o a seu estado original.
